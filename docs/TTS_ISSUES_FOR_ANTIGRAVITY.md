## TTS-1: Phoneme 개별 발음 문제 (시스템적)

현재 음소(phoneme) 버튼을 누르면 영어 발음이 아닌 이상한 발음이 나옴.

| 음소 | 현재 발음 | 올바른 발음 | 비고 |
|------|----------|-----------|------|
| `n` | "엔엔엔" (letter name 반복) | /n/ (짧은 비음) | 비슷한 케이스 모두 점검 필요 |
| `s` + `it` | "써" + "아이트" | /s/ + /ɪt/ | 이전에 개선했다고 했으나 여전히 문제 |
| `ed` | 이상한 발음 | /ɛd/ | b+ed, r+ed 등에서 확인 |
| `r` | 영어 발음 아닌 듯 | /ɹ/ | 이질감 심함 |

### 조치 필요
- 모든 onset/rime phoneme 오디오 파일 전수 점검
- Letter name이 아닌 phoneme sound로 재생성
- ElevenLabs TTS로 일괄 교체

## TTS-2: 음소 조합 발음 어색

Blend & Tap 단계에서 onset + rime 조합 발음이 어색함.

| 조합 | 문제 |
|------|------|
| `m` + `an` | 두 발음 다 어색 |
| `m` + `ap` | 두 발음 다 어색 |
| `b` + `ed` | "ed" 부분 발음 이상 |
| `r` + `ed` | 둘 다 영어 발음이 아닌 듯, 이질감 |

### 조치 필요
- 위 조합들의 onset/rime 오디오 파일 재생성
- Antigravity가 한꺼번에 일괄 처리

## 우선순위
1. **긴급**: phoneme 개별 발음 (n, r, ed 등) — 학습 경험 직접 영향
2. **높음**: 음소 조합 발음 (m+an, m+ap 등) — Blend & Tap 핵심 기능
3. **보통**: 개별 단어 발음 (pan, fed) — 특정 단어만 해당
4. **전수조사**: Word Gallery 레이첼 목소리 녹음파일 확인
