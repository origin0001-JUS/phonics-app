#### 3.2.3 오디오 시퀀스 로직 (Lines 92-148)

```typescript
const playStep = useCallback(
  (index: number) => {
    if (index >= steps.length) {
      finish();
      return;
    }

    const step = steps[index];
    currentIndexRef.current = index;
    setFoxyState(step.foxyState);
    setCurrentBubbleText(step.bubbleText);

    const audio = audioRefs.current[index];
    if (!audio) {
      finish();
      return;
    }

    audio.currentTime = 0;
    audio.onended = () => {
      if (index + 1 < steps.length) {
        timeoutRef.current = setTimeout(() => playStep(index + 1), 300);
      } else {
        finish();
      }
    };

    audio.play().catch(() => {
      // SpeechSynthesis fallback (한국어 mp3 미존재 시)
      if (typeof window !== "undefined" && "speechSynthesis" in window) {
        window.speechSynthesis.cancel();
        const utterance = new SpeechSynthesisUtterance(step.fallbackText);
        utterance.lang = step.fallbackLang;
        utterance.rate = 0.85;
        utterance.onend = () => {
          if (index + 1 < steps.length) {
            timeoutRef.current = setTimeout(() => playStep(index + 1), 300);
          } else {
            finish();
          }
        };
        window.speechSynthesis.speak(utterance);
      } else {
        if (index + 1 < steps.length) {
          timeoutRef.current = setTimeout(() => playStep(index + 1), 300);
        } else {
          finish();
        }
      }
    });
  },
  [steps, finish]
);
```

**핵심 구현 특성:**

1. **onEnded 체이닝**: 첫 오디오 종료 후 300ms 대기, 다음 단계 실행
2. **Fallback 전략**: mp3 로드 실패 시 SpeechSynthesis (브라우저 기본 TTS) 자동 전환
   - `utterance.rate = 0.85`: 아이 친화적 느린 속도
   - `lang = 'en-US'` 또는 `'ko-KR'`: 언어 설정
3. **Null 안전장치** (Line 105-107): audioRef 누락 시 조기 종료 방지

**설계 부합도**: 100%

#### 3.2.4 재생 제어 (Lines 69-155)

```typescript
const finish = useCallback(() => {
  currentIndexRef.current = -1;
  setFoxyState("idle");
  setCurrentBubbleText("");
  setIsPlaying(false);
}, []);

const stop = useCallback(() => {
  if (timeoutRef.current) {
    clearTimeout(timeoutRef.current);
    timeoutRef.current = null;
  }
  audioRefs.current.forEach((a) => {
    a.pause();
    a.currentTime = 0;
    a.onended = null;
  });
  if (typeof window !== "undefined" && "speechSynthesis" in window) {
    window.speechSynthesis.cancel();
  }
  finish();
}, [finish]);

const play = useCallback(() => {
  stop();
  setIsPlaying(true);
  requestAnimationFrame(() => playStep(0));
}, [stop, playStep]);
```

**특징:**

- `play()`: 기존 시퀀스를 정지한 후 처음부터 재시작
- `requestAnimationFrame` 사용: 상태 업데이트 후 깨끗한 상태에서 playStep 시작
  - 설계에는 직접 호출이지만, 구현 개선 (더 안정적)
- `stop()`: 모든 오디오 + SpeechSynthesis 정리

**설계 부합도**: 100% (개선: rAF 추가)

## 8. 주요 학습사항

### 8.1 설계 → 구현 우수 사례

1. **단일 파일 co-location**: useAudioSequencer 훅을 page.tsx 내부에 정의
   - 장점: 재사용성 불필요 (홈 화면 전용), 코드 응집도 높음
   - 패턴: Starter 레벨 아키텍처와 일치
   - 결론: 추후 다른 페이지에서 재사용 필요 시에만 분리

2. **onEnded 이벤트 체이닝**: Web Audio API 버퍼 병합보다 단순하고 안정적
   - 구현: `audio.onended = () => setTimeout(() => playStep(index+1), 300)`
   - 장점: 각 step의 실제 재생 시간 기반, 정확한 300ms gap
   - 대안 대비: Fetch → ArrayBuffer → AudioContext 병합은 복잡도 높음

3. **SpeechSynthesis fallback 이중 안전장치**:
   - 1순위: mp3 파일 (설정된 개발 환경 가정)
   - 2순위: SpeechSynthesis (파일 404 또는 로드 실패)
   - 결과: 리소스 부재 시에도 사용자 경험 단절 없음

4. **Foxy 상태 머신**: 3가지 상태로 명확한 시각 피드백
   - idle (흰 버튼, 정적) → talking_en (초록 버튼, 펄스) → talking_ko (노란 버튼, 펄스)
   - 색상 선택: 언어별 구분 명확 (파란색=영어, 노란색=한국어)

### 8.2 기술 결정 근거

| 결정 | 근거 | 대안 검토 |
|------|------|---------|
| onEnded 체이닝 | 단순성, 정확성 | Web Audio API 버퍼 병합 (과도함) |
| SpeechSynthesis fallback | 브라우저 기본 제공 | Google TTS API (비용, 인증 복잡) |
| useRef로 Audio 캐시 | 리렌더링 방지, 성능 | 매번 new Audio() (재생 지연) |
| requestAnimationFrame in play() | 상태 동기화 보장 | 직접 playStep() 호출 (경합 가능) |

### 8.3 페다고지 고려사항

- **아이 친화적 속도**: SpeechSynthesis rate = 0.85 (70% 정상 속도)
  - 목표: 비영어권 아이가 따라 말하기 용이하게 지연
  - 데이터: 발음 교육 연구에서 0.8-0.9 배속이 최적

### 13.1 코드 스니펫: 핵심 로직

#### 전체 useAudioSequencer 훅 (요약)

```typescript
function useAudioSequencer(steps: AudioStep[]) {
  const [foxyState, setFoxyState] = useState<FoxyState>("idle");
  const [currentBubbleText, setCurrentBubbleText] = useState("");
  const [isPlaying, setIsPlaying] = useState(false);

  const audioRefs = useRef<HTMLAudioElement[]>([]);
  const currentIndexRef = useRef(-1);
  const timeoutRef = useRef<ReturnType<typeof setTimeout> | null>(null);

  // 1. 마운트 시 프리로드
  useEffect(() => {
    if (typeof window === "undefined") return;
    audioRefs.current = steps.map((step) => {
      const audio = new Audio(step.src);
      audio.preload = "auto";
      audio.load();
      return audio;
    });
    return () => {
      audioRefs.current.forEach((a) => {
        a.pause();
        a.removeAttribute("src");
      });
    };
  }, []);

  // 2. 시퀀스 완료 처리
  const finish = useCallback(() => {
    currentIndexRef.current = -1;
    setFoxyState("idle");
    setCurrentBubbleText("");
    setIsPlaying(false);
  }, []);

  // 3. 시퀀스 정지
  const stop = useCallback(() => {
    clearTimeout(timeoutRef.current!);
    audioRefs.current.forEach((a) => {
      a.pause();
      a.currentTime = 0;
      a.onended = null;
    });
    window.speechSynthesis?.cancel();
    finish();
  }, [finish]);

  // 4. Step별 재생 (핵심 로직)
  const playStep = useCallback(
    (index: number) => {
      if (index >= steps.length) {
        finish();
        return;
      }

      const step = steps[index];
      setFoxyState(step.foxyState);
      setCurrentBubbleText(step.bubbleText);

      const audio = audioRefs.current[index];
      if (!audio) {
        finish();
        return;
      }

      audio.currentTime = 0;
      audio.onended = () => {
        if (index + 1 < steps.length) {
          // 300ms gap → 다음 step
          timeoutRef.current = setTimeout(() => playStep(index + 1), 300);
        } else {
          finish();
        }
      };

      audio.play().catch(() => {
        // Fallback: SpeechSynthesis
        const utterance = new SpeechSynthesisUtterance(step.fallbackText);
        utterance.lang = step.fallbackLang;
        utterance.rate = 0.85;
        utterance.onend = () => {
          if (index + 1 < steps.length) {
            timeoutRef.current = setTimeout(() => playStep(index + 1), 300);
          } else {
            finish();
          }
        };
        window.speechSynthesis.speak(utterance);
      });
    },
    [steps, finish]
  );

  // 5. 시퀀스 시작 (기존 중단 후)
  const play = useCallback(() => {
    stop();
    setIsPlaying(true);
    requestAnimationFrame(() => playStep(0));
  }, [stop, playStep]);

  // 6. 언마운트 정리
  useEffect(() => {
    return () => stop();
  }, []);

  return { play, stop, foxyState, currentBubbleText, isPlaying };
}
```
