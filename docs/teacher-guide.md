# 소리로 읽는 영어 300 — 교사용 가이드

> **대상**: 초등 1~4학년 담당 교사 / 방과후 영어 강사
> **앱 버전**: Phonics 300 PWA
> **수업 시간**: 유닛당 10분 (하루 1~2유닛 권장)

---

## 1. 앱 소개

**소리로 읽는 영어 300**은 한국 초등학생을 위한 파닉스 학습 앱입니다.

- **340+개 필수 어휘**를 체계적으로 학습
- **6단계 수업 흐름**으로 10분 완성 레슨
- **SM-2 간격 반복** 알고리즘으로 자동 복습 관리
- **게임화 요소** (트로피, 배지)로 학습 동기 부여
- **오프라인 지원** — 인터넷 없이도 사용 가능 (PWA)
- **개인정보 보호** — 서버 전송 없음, 기기 내 저장

## 2. 커리큘럼 구조

### 전체 구성 (37 유닛)

| 레벨 | 유닛 | 내용 | 대상 |
|------|------|------|------|
| **CoreA** | 1~6 | 단모음 (Short Vowels) + 복습 | 1~2학년 |
| **CoreA** | 7~12 | 장모음 (Long Vowels) + 복습 | 1~2학년 |
| **CoreB** | 13~18 | 자음 블렌드 & 이중자음 + 복습 | 2~3학년 |
| **CoreB** | 19~24 | 고급 패턴 (R-통제, 이중모음) + 복습 | 2~3학년 |
| **L3** | 25~30 | 자음군 & 이중자음 심화 | 3~4학년 |
| **L4** | 31~37 | 모음팀 & R-통제 모음 심화 | 3~4학년 |

### 진행 순서
- 유닛은 **순서대로 잠금 해제** (이전 유닛 완료 필수)
- 복습 유닛(6, 12, 18, 24)은 이전 5개 유닛 모두 완료 후 해제
- Exit Ticket **80% 이상** 정답 시 다음 유닛으로 진행

## 3. 수업 흐름 (10분 레슨)

각 유닛은 6단계로 구성됩니다:

### Step 1: Sound Focus (2분)
**목표**: 목표 소리 인식 및 발음

- 교사가 목표 소리를 시범 발음
- 입모양 시각화(MouthVisualizer)로 정확한 조음 위치 확인
- 학생이 따라 발음
- **교사 팁**: 한국어 유사음과 비교하여 설명

### Step 2: Blend & Tap (2분)
**목표**: 음소 블렌딩 연습

- 음소를 하나씩 탭하며 합치기 (예: /k/-/æ/-/t/ → cat)
- 손가락으로 책상 탭핑 — 멀티센서리 접근
- 3~4개 단어 연습
- **교사 팁**: 천천히 시작하여 점점 빠르게

### Step 3: Decode Words (2분)
**목표**: Onset/Rime 분리 및 디코딩

- 화면의 단어를 onset(첫소리)과 rime(끝소리)으로 분리
- 예: c + at = cat, b + at = bat
- Word Family 패턴 인식 강화
- **교사 팁**: 같은 rime을 가진 단어끼리 묶어서 연습

### Step 4: Say & Check (2분)
**목표**: 발음 확인 및 교정

- 학생이 단어를 소리 내어 말하기
- 음성 인식으로 정확도 확인
- 정확하면 다음 단어, 부정확하면 재시도
- **교사 팁**: 조용한 환경에서 진행, 마이크 가까이

### Step 5: Micro-Reader (1분)
**목표**: 문맥 속 단어 읽기

- 배운 단어가 포함된 짧은 문장 읽기
- 유창성(fluency) 연습
- **교사 팁**: 먼저 교사가 읽고, 학생이 따라 읽기

### Step 6: Exit Ticket (1분)
**목표**: 학습 확인

- 4~5개 단어 퀴즈
- 80% 이상 → 유닛 완료, 다음 유닛 해제
- 80% 미만 → 재도전 또는 복습 권장

## 5. 교수법 가이드

### 멀티센서리 접근법
모든 수업에서 3가지 감각을 활용합니다:
- **시각**: 글자, 그림, 입모양 시각화
- **청각**: TTS 음성, 교사 시범 발음
- **운동감각**: 책상 탭핑, 화면 터치

### Word Family 교수법
같은 rime(-at, -an 등)을 가진 단어를 묶어 가르칩니다:
1. 기본 rime 소개 (-at)
2. onset 교체하며 새 단어 생성 (c+at, b+at, h+at)
3. 패턴 인식 → 새 단어 자가 디코딩 가능

### 한국어 연결 전략
| 영어 소리 | 한국어 유사음 | 차이점 |
|-----------|-------------|--------|
| /æ/ (short a) | "애" | 입을 더 크게 벌림 |
| /ɛ/ (short e) | "에" | 거의 동일 |
| /ɪ/ (short i) | "이" | 더 짧고 약함 |
| /ɒ/ (short o) | "오" | 입을 더 크게 벌림 |
| /ʌ/ (short u) | "어" | 입을 약간만 벌림 |
| /θ/ (th 무성) | 없음 | 혀를 이 사이에 넣고 바람 |
| /ð/ (th 유성) | 없음 | 같은 위치에서 성대 진동 |

### 복습 시스템 (SM-2)
- 앱이 **자동으로** 복습 시기를 계산합니다
- 홈 화면에 "복습할 단어 N개" 배지 표시
- 매일 앱 시작 시 복습 먼저 진행 권장
- 단계: 처음 → 1일 후 → 3일 후 → 7일 후 → 14일 후 → 30일 후

## 8. 자주 묻는 질문 (FAQ)

**Q: 인터넷 없이 사용할 수 있나요?**
A: 네, PWA로 설치하면 오프라인에서도 모든 기능이 동작합니다.

**Q: 학생 데이터가 외부로 전송되나요?**
A: 아니요. 모든 데이터는 기기 내 IndexedDB에 저장되며 서버 전송이 없습니다.

**Q: 한 기기에서 여러 학생이 사용할 수 있나요?**
A: 현재 버전은 기기당 1명 기준입니다. 설정에서 데이터 초기화 후 다른 학생이 사용할 수 있습니다.

**Q: 학생이 Exit Ticket을 통과하지 못하면?**
A: 같은 유닛을 다시 학습할 수 있습니다. SM-2 알고리즘이 틀린 단어를 더 자주 복습하도록 스케줄링합니다.

**Q: 소리가 안 나요.**
A: 기기 볼륨 확인 → 브라우저 자동재생 정책 때문에 처음에 화면을 한 번 터치해야 합니다.
