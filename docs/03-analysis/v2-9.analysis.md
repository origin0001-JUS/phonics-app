**Notes**:
- Design says "300 words" but ~200+ PNG files currently exist. Remaining words (especially L3/L4 units 25-37, ~170 words from `l3l4Words.ts`) may not have images yet. The `WordImage` component gracefully handles missing images via `onError={() => setError(true)}` returning `null`, so this is non-blocking.
- `generate-images-gemini.ts` uses `getAllWords()` from curriculum.ts (covers all ~336 words) and skips already-existing files, so re-running the script will fill gaps.
- QA verification script (`verify-images.ts`) is a valuable addition beyond spec -- uses Gemini vision to validate each image matches its word.
